# FUTURE_ML_01 – Machine Learning Projects

##  OVERVIEW

This repository contains all tasks completed as part of the **Future Interns Machine Learning Internship**.
The projects demonstrate practical implementation of machine learning techniques across different domains.

---

## PROJECT STRUCTURE

### Task 1: Sales Forecasting

* Predict future sales using historical data
* Uses regression-based machine learning models

**Files:**

* `data.csv` – Raw dataset
* `final_forecast.csv` – Predicted results
* `sales_forecast.ipynb` – Model implementation

---

### Task 2: Chatbot Classification

* Classifies customer support tickets into categories
* Uses NLP techniques for text processing

**Files:**

* `customer_support_tickets.csv` – Dataset
* `CustomerSupport.py` – Classification model
* Output screenshots included

---

### Task 3: Resume Screening System

* Automates resume classification and ranking
* Helps shortlist candidates efficiently

**Files:**

* `Resume_small.csv` – Dataset (optimized for GitHub)
* `app.py` – Main application
* `Resume_Screening_Demo.ipynb` – Model workflow
* `ranking.csv` – Output results
