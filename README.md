# FUTURE_ML_01 – Sales Forecasting Project

## OVERVIEW

This project focuses on analyzing historical sales data and generating future sales forecasts using machine learning techniques.

## FILES INCLUDED

* `data.csv` – Raw dataset
* `final_forecast.csv` – Predicted results
* `sales_forecast.ipynb` – Jupyter Notebook with complete implementation

## TECHNOLOGIES USED

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

## OBJECTIVE

To build a model that can predict future sales trends based on historical data.

## OUTPUT

The model generates forecasted sales values stored in `final_forecast.csv`.
